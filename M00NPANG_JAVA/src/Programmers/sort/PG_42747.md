### 프로그래머스_Sort_H-Index

#### 결론 : 처음에 H-Index가 너무 이해가 안 가서리... 오래 걸렸는데 위키백과 최고다!

1. 논문의 인용 횟수를 오름차순 정렬
2. 정렬된 배열에서 왼쪽부터 하나씩 확인하면서 H-Index 조건을 만족하는 값을 확인
3. 만족하는 조건 반환
4. 조건 만족하지 않으면 0 반환


``` java
import java.util.*;

class Solution {
    public int solution(int[] citations) {
        // 인용 횟수 내림차순 정렬
        Arrays.sort(citations);
        int n = citations.length;

        // H-Index 계산
        for (int i = 0; i < n; i++) {
            int h = n - i; 
            if (citations[i] >= h) {
                return h;
            }
        }

        return 0;
    }
}

```
